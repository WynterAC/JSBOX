# session.sftp

sftp instance in session object provides file related APIs, object properties:

Prop | Type | Description
---|---|---
session | session | session
bufferSize | number | buffer size
connected | bool | is connected

# sftp.connect()

Create SFTP connection:

```js
await sftp.connect();
```

---|---|---
