> Track sensor data updates

# $motion.startUpdates(object)

Start updates:

```js
$motion.startUpdates({
  interval: 0.1,
  handler: function(resp) {

  }
})
```

Learn more about the data [CMDeviceMotion](https://developer.apple.com/documentation/coremotion/cmdevicemotion).

# $motion.stopUpdates()

Stop updates.
